# Sizing og spacing

Vi bruker tokens for å definere spacing (mellomrom) og sizing (bredder og høyder) på en konsistent måte. Vi benytter skalaverdier (f.eks. `size5`, `size6`, `size10`) som er relative, men som i praksis oversettes til faste verdier i `rem`. Dette gjør systemet fleksibelt og responsivt.

> 1rem tilsvarer 16px i vår standardkonfigurasjon (`data-size="medium"`).

Spacing-tokenene bygger på en skala som øker jevnt, og kan brukes til alt fra innholdsmarginer til komponent-avstand og layout-bredder.
