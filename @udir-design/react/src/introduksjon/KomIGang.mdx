# Kom i gang

Her finner du det du trenger for å installere og ta i bruk Udirs komponentbibliotek i et prosjekt.

## Installer pakkene

Designsystemet består av flere pakker med ulike formål. For å bruke komponentbiblioteket må du installere `@udir-design/react` og `@udir-design/theme`.

- [`@udir-design/react`](https://github.com/Utdanningsdirektoratet/designsystem/tree/main/%40udir-design/react) - React-komponenter og tilhørende funksjonalitet.
- [`@udir-design/theme`](https://github.com/Utdanningsdirektoratet/designsystem/tree/main/%40udir-design/theme) - Design tokens for blant annet farger, typografi, spacing og størrelser.

```bash
npm add @udir-design/react @udir-design/theme
```

Andre pakker i designsystemet kan også være relevante, avhengig av behovene i prosjektet ditt:

- [`@udir-design/css`](https://github.com/Utdanningsdirektoratet/designsystem/tree/main/%40udir-design/css) - CSS-filer og stilsett for designsystemet.
- [`@udir-design/icons`](https://github.com/Utdanningsdirektoratet/designsystem/tree/main/%40udir-design/icons) - Ikoner tilgjengelig som React-komponenter.
- [`@udir-design/symbols`](https://github.com/Utdanningsdirektoratet/designsystem/tree/main/%40udir-design/symbols) - Symboler og grafiske elementer brukt i Udirs tjenester.

```bash
npm add @udir-design/css @udir-design/icons @udir-design/symbols
```

## Last inn font

Komponentbiblioteket bruker fonten `Inter`. For at komponentene skal vises med riktig typografi, må fonten lastes inn i applikasjonen.

Legg følgende i `<head>` i `index.html`:

```html
<link
  rel="preload stylesheet"
  as="style"
  href="https://altinncdn.no/fonts/inter/v4/inter.css"
/>
```

## Last inn stilsettet

For at komponentene skal vises korrekt, må du laste inn bibliotekets stilsett fra `@udir-design/react/style.css`.

Du kan gjøre dette i JavaScript- eller TypeScript-entrypointet ditt:

```tsx
import '@udir-design/react/style.css';
```

Alternativt kan du importere stilsettet fra en CSS-fil:

```css
@import '@udir-design/react/style.css';
```

## Konfigurer typografi

Biblioteket setter ingen globale stiler. For å få en konsistent typografi på innhold som ikke bruker komponenter fra biblioteket, anbefaler vi å sette en standard typografistil på `body`:

```css
body {
  font-family: 'Inter', sans-serif;
  font-feature-settings: 'cv05' 1; /* Enable lowercase l with tail */
}
```

## Legg til TypeScript-typer

Bruker du TypeScript, anbefaler vi å legge til typene for `@udir-design/theme` og `@udir-design/react/html` i `tsconfig.json`. Da får du typesjekking og autocomplete for Udirs farger og størrelser (`data-color` og `data-size`-attributter), både på komponenter fra biblioteket og vanlige HTML-elementer.

Legg til følgende i `compilerOptions`:

```json
{
  "compilerOptions": {
    "types": ["@udir-design/theme", "@udir-design/react/html"]
  }
}
```

## Ta i bruk komponentene

Når oppsettet er på plass, kan du importere og bruke komponentene i applikasjonen din.

```tsx
import { Details, Card } from '@udir-design/react';
import { ChevronDownIcon } from '@udir-design/icons';

export const MyComponent = () => (
  <Card>
    <Details>
      <Details.Summary>
        <ChevronDownIcon aria-hidden />
        Første overskrift
      </Details.Summary>
      <Details.Content>Skjult innhold som kan åpnes</Details.Content>
    </Details>
    <Details>
      <Details.Summary>
        <ChevronDownIcon aria-hidden />
        Andre overskrift
      </Details.Summary>
      <Details.Content>Mer skjult innhold her</Details.Content>
    </Details>
  </Card>
);
```

Du finner oversikt over alle komponentene og tilhørende dokumentasjon i menyen til venstre.
