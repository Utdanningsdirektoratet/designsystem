# Ikoner

Ikoner kan brukes for å raskt identifisere interaktive elementer og skille mellom kategorier. Ikonene er tilgjengelig både som React-komponenter og som svg-filer. Vi anbefaler å bruke React-komponentene når du bruker React.

Vi følger [Digdirs anbefaling om å bruke Navs ikonbibliotek Aksel](https://design.udir.no/main/?path=/docs/iconsandsymbols-retningslinjer--docs#foretrukket-hos-udir). Noen ikoner, slik som piler, finnes i ulike varianter. For disse har vi valgt ut noen som er merket "Foretrukket hos Udir" slik at vi bruker ikoner mest mulig konsistent på tvers i tjenestene våre. Vi bruker primært outline-versjon av ikonene, men i enkelttilfeller vil det fungere med filled.
