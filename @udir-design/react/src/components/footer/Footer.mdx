# Footer

`Footer` brukes nederst på siden for viktig informasjon og lenker.

**Passer til å**

- lenke til tilgjengelighetserklæring

**Passer ikke til å**

- bruke som en "ekstra" meny

## Slik bruker du Footer

<Controls />

import { Footer } from '@udir-design/react';

<Footer>Lenke</Footer>;
