# Slik støtter du flere språk

Udirs designsystem støtter flerspråklige løsninger. I de fleste komponentene håndterer du språkvalg selv ved å sende inn tekst som props eller children.

Noen komponenter har tekst som er forhåndsdefinert. Disse komponentene vil automatisk bruke riktig språk basert på hva [`lang`-attributtet](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Global_attributes/lang) i løsningen er satt til. Støttede språk er `"nb"` (norsk bokmål) eller `"en"` (engelsk). Det vil fungere å sette `lang` på hvilken som helst forelder, men gjerne sett det på `<html>` for å sikre konsistens.

```jsx
<div lang="en">
  <FileUpload />
</div>
```

Ønsker du å støtte flere språk i disse komponentene, kan du overstyre standardtekstene i CSS-variabler. Det gjør du ved å overstyre variablene i `lang`-attributtet for ønsket språk:

```css
[lang='se'] {
  --udsc-fileUpload-uploading-text: 'Viežžamin…';
  --udsc-fileUpload-addFile-text: 'Lasit fiilla';
}
```
