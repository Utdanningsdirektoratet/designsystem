# Badge

`Badge` er en ikke-interaktiv komponent som viser status med eller uten tall. Bruk [Tag](/docs/components-tag--docs) dersom du skal
ha tekst i stedet for tall.

<Primary />
<Controls />

## Bruk

```tsx
import { Badge } from '@udir-design/react';

/* Inline */
<Badge />

/* Flytende over element */
<Badge.Position>
  <Badge />
  <Elem />
</Badge.Position>
```

## Flytende over element

Sett plassering med `placement`. Dersom elementet du plasserer over er en sirkel kan du sette `overlap="circle"`.

<Canvas of={BadgeStories.Floating} />

## Farger

`Badge` kommer i alle farger, du kan endre den med `data-color`.
Du kan også sette `data-variant` direkte for å endre bakgrunnsfarge.

<Canvas of={BadgeStories.Variants} />

## Vise staus

Dersom du ikke sender inn `count`, vises kun en sirkel. Dette kan brukes for å vise status.

<Canvas of={BadgeStories.Status} />

## I tabs

Du kan bruke komponenten inline eller flytene over et element.

<Canvas of={BadgeStories.InTabs} />

## I button

Du kan bruke komponenten inni eller flytende på en knapp.
