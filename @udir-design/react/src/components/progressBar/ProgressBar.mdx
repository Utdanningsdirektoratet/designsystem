# ProgressBar

`ProgressBar` viser fremgang i en prosess. Den kan for eksempel brukes i skjemaer med flere sider for å vise hvor mange sider som er fullført eller for å vise fremgang ved opplasting eller nedlasting av innhold.

**Passer til å**

- brukes i skjemaer uten gruppering der spørsmålene er frivillige
- vise fremgang ved opplasting eller nedlasting av innhold

**Passer ikke til å**

- brukes i skjemaer med validering, bruk heller [`FormProgress`](/link/kommer/)

## Slik bruker du ProgressBar

<Primary />
<Controls />

```tsx
import { ProgressBar } from '@udir-design/react';

<ProgressBar current={2} max={10} />;
```
