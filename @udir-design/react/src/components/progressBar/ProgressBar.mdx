# ProgressBar

`ProgressBar` viser fremgang i enkle skjemaer.

**Passer til å**

- brukes i skjemaer uten gruppering der spørsmålene er frivillige

**Passer ikke til å**

- brukes i skjemaer med validering, bruk heller [`FormProgress`](/link/kommer/)

## Slik bruker du ProgressBar

<Primary />
<Controls />

```tsx
import { ProgressBar } from '@udir-design/react';

<ProgressBar current={2} max={10} />;
```
