# FileUpload

`FileUpload` er et sett med delkomponenter som brukes når det skal lastes opp filer.

**Passer til å**

- brukes til opplasting av filer
- å gi brukeren muligheten til å vise filen før innsending
- å vise eventuelle feilmeldinger

**Passer ikke til å**

- håndtere opplasting av sensitive filer uten ekstra sikkerhetstiltak
- gjennomføre alidering og virussjekk av filer

## Slik bruker du FileUpload

<Primary />
<Controls />

```tsx
import { FileUpload } from '@udir-design/react';

/* Egen logikk */

<FileUpload />;
```

## Eksempler

Du må [selv håndtere input av filer](https://developer.mozilla.org/en-US/docs/Web/HTML/Reference/Elements/input/file) med `input` slik som i eksemplene under.

### Knapp

Ved bruk av varianten `Compact` har man ikke samme behov for et ekstern bibliotek. Bruk heller propertien `onChange` til å håndtere endringer i `<input>`-feltet.

### Vedlegg

Delkomponenten `FileUpload.Item` brukes til å vise status for filer som er lastet opp. Du kan bruke `error` til å vise feilmeldinger, f.eks. dersom filformatet ikke er riktig.
