# Chip

`Chip` er ein liten, interaktiv komponent som lèt brukarane styre korleis dei vil sjå innhald. For eksempel kan den brukast til å filtrere kategoriar i eit søkeresultat og vise kva for filter som er aktive.

**Passar til å:**

- Filtrere informasjon, for eksempel i lister eller tabellar.
- Vise kva for filter som er valde.

**Passar ikkje til:**

- Varsel for tal eller status (bruk heller [`Badge`](/docs/components-badge--docs)).
- Å merke innhald med kategori/emne som ikkje er interaktivt. Bruk heller [`Tag`](/docs/components-tag--docs)-komponenten dersom kategori/emne ikkje er klikkbart.
- Menyar, som sidemenyar eller hovudmenyar

## Slik brukar du Chip

<Primary />
<Controls />

```tsx
import { Chip } from '@udir-design/react';

// som radio
<Chip.Radio name="my-chip" defaultChecked>
  Bokmål
</Chip.Radio>;

// som checkbox
<Chip.Checkbox name="my-chip" defaultChecked>
  Barnehage
</Chip.Checkbox>;

// som removable
<Chip.Removable>Grunnskole</Chip.Removable>;

// som button
<Chip.Button>Vidregående</Chip.Button>;
```

## Varianter

### `Chip.Checkbox`

`Chip.Checkbox` kan brukast som eit alternativ til `Checkbox`.

<Canvas of={ChipStories.Checkbox} />

### `Chip.Removable`

`Chip.Removable` inneheld eit kryss som indikerer at filteret kan fjernast. Det er viktig å merke seg at `aria-label` må leggjast til dersom innhaldet i Chip ikkje forklarer at den kan fjernast. I dette eksempelet er det lagt til `aria-label="Slett {land}"`.

<Canvas of={ChipStories.Removable} />

### Chip.Button

`Chip.Button` kan brukes i gruppe, for å vise valgmuligheter - ikke handlinger.

<Canvas of={ChipStories.Button} />

## Retningslinjer

`Chip` lèt brukarane styre korleis dei vil vise innhald.

Unngå å plassere `Chip` vertikalt. Skal du vise alternativ under kvarandre, bruk heller [`Checkbox`](/docs/components-checkbox--docs) eller [`Radio`](/docs/components-radio--docs). En gruppe med flere `Chip` kan brytast over fleire linjer.

## Tekst

`Chip` bør ha så få ord som mogleg, helst berre eitt eller to.

## Tilgjengelegheit

Viss delar av ei side blir oppdaterte samtidig som brukarane gjer noko på sida, må vi passe på at brukarar med skjermlesarar får beskjed om det. Vi kan bruke ein `ARIA live region` til det, som kan vise tal på treff eller andre endringar.
