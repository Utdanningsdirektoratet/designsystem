# Komponenter

Bruk venstremenyen for å navigere i dokumentasjonen.

Overskriften "Demosider" inneholder eksempler som viser komponenter brukt sammen. Alle individuelle komponenter finner du under "Komponenter".

Komponenter er markert med "Alpha" eller "Beta" dersom de ikke er stabile (les mer om livsfaser lenger ned).

## Kommende komponenter

Noen av våre systemer har behov som ikke dekkes av komponentene i Digdirs designsystem. Vi har notert behov for og besluttet å implementere følgende komponenter:

- Header
- Footer
- Stepper
- HelpText
- TableOfContents
