# Systemvarsler

Systemvarsler informerer brukerne om feil i systemet, eller viktige hendelser de bør være oppmerksomme på.

Udir følger [bruksmønster for systemvarsler utarbeidet i Digdirs 'Designsystemet'](https://designsystemet.no/no/patterns/systemnotifications).
