# Hjelpefunksjoner for datavisualisering

Verktøy for å bruke datavisualiseringsfarger i diagrammer og grafer. Funksjonene returnerer hex-fargeverdier som fungerer direkte i alle diagrambiblioteker og stilark.

- `getCategoricalColors()`
- `generateSequentialMonochromaticColors(count: number)`
- `generateSequentialDivergentColors(count: number)`
- `getHighchartsTheme()`

Les også <Link href="./typedoc/modules/utilities_datavis_alpha.html">detaljert dokumentasjon generert fra kildekoden (på engelsk)</Link>.

## Oppsett

Importer CSS for datavisualiseringsfarger i applikasjonen din:

```tsx
import '@udir-design/theme/datavis.css';
```

## Kategorisk palett

For distinkte kategorier i diagrammer (fagområder, elevgrupper, etc.).

**`getCategoricalColors()`** - Returnerer 8 CSS-fargeverdier.

```tsx
import { getCategoricalColors } from '@udir-design/react/utilities/alpha';

const colors = getCategoricalColors();
// ['var(--uds-data-color-categorical-1)', 'var(--uds-data-color-categorical-2)', ...]
```

## Sekvensiell monokromatisk palett

For sekvensiell data med gradert skala fra lys til mørk (heatmaps, choropleth-kart).

**`generateSequentialMonochromaticColors(count: number)`** - Genererer et egenvalgt antall hex-fargeverdier (minimum 1).

```tsx
import { generateSequentialMonochromaticColors } from '@udir-design/react/utilities/alpha';

const colors = generateSequentialMonochromaticColors(7);
// ['#7edfae', '#75d0a2', '#6dc297', ...]
```

<Unstyled>
  <DatavisGenSeqColorDisplay />
</Unstyled>

For å bruke fargene som CSS-variabler, kan du kopiere koden over og legge den i ditt eget stilark. Det gjør det enkelt å bruke fargene i diagrammer og grafer som støtter CSS-variabler.

---

## Sekvensiell divergerende palett

For divergerende skalaer med nøytralt midtpunkt (avvik fra gjennomsnitt).

**`generateSequentialDivergentColors(count: number)`** - Genererer et egenvalgt antall hex-fargeverdier (minimum 3). Paletten deles i to armer (grønn og blå) med et nøytralt midtpunkt som alltid er med i resultatet.

```tsx
import { generateSequentialDivergentColors } from '@udir-design/react/utilities/alpha';

const colors = generateSequentialDivergentColors(7);
// ['#1e422f', '#294f3a', ..., '#dedede', ..., '#3a4d68', '#2e3c51']
```
